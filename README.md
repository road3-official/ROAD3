# ROAD3

Version 1.0.0

ROAD3は、「自由」と「安心」を両立することを目指したSNSです。

ユーザーが自分だけの居場所（パーソナルスペース）を持ち、
そこからオープン・グループ・クローズの各スペースへ自由に行き来できる、
新しいSNSを目指して開発しています。

---

## 主な機能

- パーソナルスペース
- クローズスペース（DM）
- グループスペース
- オープンスペース
- ログイン・新規登録
- 利用規約・プライバシーポリシー
