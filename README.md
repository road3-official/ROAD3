# ROAD3

Version 1.0.0

ROAD3は、自由と安心を両立することを目指したSNSです。

## 主な機能

- パーソナルスペース
- クローズスペース
- グループスペース
- オープンスペース
- ログイン・新規登録
- 利用規約・プライバシーポリシー
